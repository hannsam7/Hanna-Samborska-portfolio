# Hanna Samborska – Portefølje

En lettvekts, responsiv og miljøvennlig portefølje bygget med HTML, CSS og JavaScript.

## Funksjoner
- Papirlignende SVG-bakgrunn
- Moderne, animert introseksjon med typewriter-effekt
- Mørk modus med lagring av brukerens valg
- Prosjektgalleri med animasjon
- Interaktive og stilrene kontaktlenker
- Glatt scrolling og "scroll to top"-knapp
- Nedlastbar CV
- Ingen eksterne biblioteker

## Struktur
```
portfolio/
├── index.html
├── work.html
├── contact.html
├── style.css
├── script.js
└── assets/
    └── Hanna_Samborska_CV.pdf
```

## Miljøvennlige valg
- Komprimerte ressurser (ingen eksterne fonter eller bilder)
- Statisk hosting (GitHub Pages / Netlify)
- Få HTTP-forespørsler
- Lett typografi og subtile animasjoner
